Privacy Policy for Mepa

Last Updated: December 18, 2025

1. Information We Collect

Message Data: Mepa processes messages in Discord channels where it has permission. This data is transient and used to generate AI responses via the Gemini API. We do not store message content long-term.

Server Data: We store basic server settings (e.g., prefix, welcome settings) in a secure database to provide consistent functionality.

User IDs: We may store Discord User IDs for moderation logs and leaderboard features.

2. How We Use Information

We use the collected information solely to provide, maintain, and improve the Bot's features.

3. Data Sharing

We do not sell user data. Message data is shared with Google (via Gemini API) for processing AI responses. Please refer to Google's Privacy Policy for how they handle API data.

4. Data Deletion

Users can request data deletion by contacting the bot developers or removing the bot from their server.

5. Security

We use industry-standard security measures to protect stored data.
